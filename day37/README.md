# Day 37 - Go Modules (init)

## Goal
Initialize a Go project.

## Simple Explanation
A **Go Module** is how Go manages dependencies and knows where your code lives. Every project needs a `go.mod` file, which you create with `go mod init`. This file lists every library your project uses and ensures everyone working on the project uses the same versions.

## Problem Description
Create a new folder and run `go mod init <module-name>`. Research what the `go.mod` file does.
