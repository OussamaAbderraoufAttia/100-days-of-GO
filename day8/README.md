# Day 8 - While-style For Loop

## Goal
Use a for loop like a while loop.

## Simple Explanation
While Go doesn't have a `while` keyword, you can create a "while loop" by giving the `for` loop only a condition. The loop will keep running as long as that condition is true.

## Problem Description
In Go, there is no `while` keyword. Use a `for` loop with only a condition to print powers of 2 less than 100.
