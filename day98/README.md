# Day 98 - Building a Simple CRUD API (Part 2)

## Goal
Implement business logic handlers.

## Simple Explanation
Now that your project is organized, it's time to write the **Handlers**. These are the functions that actually "handle" the requests. One for getting all books, one for finding a specific book, and one for adding a new one. By keeping these in their own folder, your code stays clean and easy to test.

## Problem Description
Implement the handlers for your Book API from Day 97. Use a map as a temporary database and implement `GetAllBooks` and `CreateBook`.
