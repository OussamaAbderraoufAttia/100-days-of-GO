Deployment Strategy Summary:

1. Build a Docker Image for the application.
2. Push the image to a container registry (like Docker Hub or GitHub Container Registry).
3. Use a cloud service (AWS App Runner, Google Cloud Run, or a VPS) to pull and run the container.
4. Set up an SSL certificate (using Let's Encrypt) to ensure the site is secure (HTTPS).
5. (Advanced) Use GitHub Actions to automate steps 1-3 every time you push code.
