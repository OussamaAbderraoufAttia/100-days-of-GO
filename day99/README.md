# Day 99 - Deploying a Go App (Conceptual)

## Goal
Learn about production environments.

## Simple Explanation
Go is great for the cloud because its binaries are small and fast. You can deploy Go apps by simply copying the binary to a server (like DigitalOcean or AWS). Most people use **CI/CD** (Continuous Integration) tools which automatically build your Docker image and "ship it" to the internet whenever you save your code.

## Problem Description
Research common Go deployment strategies. Write a short summary of how you would deploy your Go application to a cloud provider.
