# 100 Days of Go

A structured learning path to take you from absolute beginner to advanced Go developer through 100 practical exercises.

## Project Structure

This repository is organized into five phases:

- **Phase 1: Go Fundamentals (1–20)** - Basic syntax, types, and control flow.
- **Phase 2: Intermediate Go (21–40)** - Collections, structs, methods, and error handling.
- **Phase 3: Go Core Concepts (41–60)** - Interfaces, file I/O, testing, and CLI tools.
- **Phase 4: Concurrency & Performance (61–80)** - Goroutines, channels, and advanced patterns.
- **Phase 5: Real Backend Development (81–100)** - HTTP servers, databases, and production-ready services.

## How to used this curriculum

1. Follow the exercises in order.
2. For each exercise, create a new directory or file in your workspace.
3. Write the solution code yourself.
4. Try the optional challenges to deepen your understanding.
5. Refer to the [Official Go Documentation](https://go.dev/doc/) whenever you get stuck.
