# Day 20 - Palindrome Checker

## Goal
Implement a basic algorithm.

## Simple Explanation
A palindrome is a word that reads the same forwards and backwards (like "radar"). To check this, we iterate through the word from both ends. We compare the first letter with the last, the second with the second-to-last, and so on. If any don't match, it's not a palindrome.

## Problem Description
Write a function that checks if a given string is a palindrome (reads the same forwards and backwards).
