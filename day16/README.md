# Day 16 - Basic String Manipulation

## Goal
Work with the strings package.

## Simple Explanation
Go's `strings` package is like a toolbox for text. You can use it to change text to uppercase, check if a sentence contains a certain word, or find the length of a string using the `len()` function. Remember that strings in Go cannot be changed once created; you always create a *new* string when you modify it.

## Problem Description
Write a program that takes a string and prints its length, converts it to uppercase, and checks if it contains a specific substring. Use the `strings` package.
