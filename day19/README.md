# Day 19 - Simple Calculator

## Goal
Combine multiple concepts into a small program.

## Simple Explanation
This project brings together input handling, conditions, and basic logic. We ask the user for two numbers and a symbol (+, -, *, /). Based on the symbol, we decide which math operation to perform.

## Problem Description
Build a command-line calculator that asks for two numbers and an operator (+, -, *, /). Use a `switch` statement to perform the operation and print the result.
