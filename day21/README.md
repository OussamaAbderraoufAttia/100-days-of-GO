# Day 21 - Arrays

## Goal
Understand fixed-size sequences.

## Simple Explanation
An array is a collection of items of the same type. In Go, the size of an array is part of its type and cannot be changed. For example, `[5]int` is an array of exactly 5 integers. We use square brackets and an index (starting at 0) to access specific elements.

## Problem Description
Declare an array of 5 integers and initialize it with values. Write a loop to calculate the sum of all elements in the array.
