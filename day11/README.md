# Day 11 - Multiple Return Values

## Goal
Return more than one value from a function.

## Simple Explanation
One of Go's coolest features is that functions can return multiple values at once. This is commonly used to return a result and an error together. You just list the return types in parentheses. When calling the function, you use a comma to catch all the results.

## Problem Description
Write a function `divide` that takes two integers. It should return the quotient and the remainder.
