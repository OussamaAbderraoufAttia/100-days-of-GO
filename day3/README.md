# Day 3 - Constants and Enumerations

## Goal
Use constants and iota.

## Simple Explanation
Constants are values that never change while the program is running. We use the `const` keyword. `iota` is a special Go tool that automatically creates a sequence of numbers (0, 1, 2, ...) starting from the top. It's great for lists like days of the week.

## Problem Description
Create a program that defines constants for the days of the week using `iota`. Print the values of these constants. Also, define a constant for the speed of light.
