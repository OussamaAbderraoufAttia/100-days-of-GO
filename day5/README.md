# Day 5 - Conditions (If/Else)

## Goal
Implement branching logic.

## Simple Explanation
We use `if` to check if something is true. For example, to check if a number is even, we use the "modulo" operator `%`. If `number % 2` is 0, it means there is no remainder, so the number is even. If it's not true, the `else` block runs.

## Problem Description
Write a program that checks if a given number is even or odd. If it's even, print "Even"; otherwise, print "Odd".
