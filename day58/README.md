# Day 58 - Basic Reflection (Type and Value)

## Goal
Inspect types at runtime.

## Simple Explanation
**Reflection** is a "superpower" that lets your code look at itself while it's running. It can find out the type of a variable or even see secret parts of a struct that it wouldn't normally know about. It's very powerful but should be used sparingly because it's slower and can be harder to understand.

## Problem Description
Use the `reflect` package to print the `Type` and `Value` of an unknown variable.
