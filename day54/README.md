# Day 54 - Sorting Basics

## Goal
Use the sort package.

## Simple Explanation
The `sort` package is Go's built-in way to put data in order. It can sort lists of whole numbers (`Ints`) or words (`Strings`). Sorting is a common task in programming, and Go's version is very, very fast.

## Problem Description
Create a slice of integers and sort them in ascending order using the `sort` package.
