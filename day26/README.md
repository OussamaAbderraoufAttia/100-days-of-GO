# Day 26 - Deleting from Maps

## Goal
Manage map content.

## Simple Explanation
To remove an item from a map, Go provides the built-in `delete()` function. You tell it which map you're using and which key you want to get rid of. If the key doesn't exist, nothing happens (it won't crash your program).

## Problem Description
Start with the map from Exercise 25. Delete one person from the map using the `delete` function and print the whole map to verify.
