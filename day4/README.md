# Day 4 - Basic Arithmetic

## Goal
Perform basic mathematical operations.

## Simple Explanation
Computing in Go uses standard symbols: `+` for adding, `-` for subtracting, `*` for multiplying, and `/` for dividing. When working with decimals, we use the `float64` type to keep things precise.

## Problem Description
Write a program that takes two floating-point numbers as variables and performs addition, subtraction, multiplication, and division. Print the results clearly.
