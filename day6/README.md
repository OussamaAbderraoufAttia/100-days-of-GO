# Day 6 - The Switch Statement

## Goal
Use switch for multiple conditions.

## Simple Explanation
A `switch` statement is a cleaner way to write many `if-else` blocks. It checks a value against several "cases". If a match is found, the code for that case runs. Go's `switch` automatically "breaks", so you don't need to write the `break` keyword like in some other languages.

## Problem Description
Create a program that takes a number (1-7) representing a day of the week and prints the name of the day. Use a `switch` statement instead of multiple `if` statements.
