# Day 10 - Basic Functions

## Goal
Define and call simple functions.

## Simple Explanation
Functions help us organize code. We define them with `func`, give them a name, and list any "parameters" (inputs) they need. If the function calculates a value, we tell it what kind of data it "returns".

## Problem Description
Write a function `multiply` that takes two integers as parameters and returns their product. Call this function from `main` and print the result.
