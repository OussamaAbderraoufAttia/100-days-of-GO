# Day 7 - For Loops

## Goal
Iterate using the only loop in Go.

## Simple Explanation
Go only has one kind of loop: the `for` loop. The standard version has three parts separated by semicolons:
1. `i := 1`: A starting point (initialization).
2. `i <= 10`: A condition (the loop stops when this is false).
3. `i++`: A step (what happens after each loop).

## Problem Description
Write a program that prints numbers from 1 to 10 using a `for` loop.
