# Day 27 - Structs

## Goal
Group related data.

## Simple Explanation
A `struct` (short for structure) lets you bundle different types of data together into one custom type. For example, a `User` struct might have a name (string) and an age (int). It's the building block for objects in Go. We use the "dot" `.` to access the internal fields.

## Problem Description
Define a `Person` struct with `Name` (string), `Age` (int), and `Email` (string). Create an instance of this struct and print its fields.
