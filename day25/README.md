# Day 25 - Maps Basics

## Goal
Use key-value pairs.

## Simple Explanation
A `map` is like a dictionary. You have a **key** (like a person's name) and a **value** (like their age). Maps are very fast for looking up information if you know the key. We create maps using the `make()` function or a literal like `map[keyType]valueType{...}`.

## Problem Description
Create a map where keys are string names and values are integer ages. Add 3 people to the map and print the age of one specific person.
